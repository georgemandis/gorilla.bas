GORILLAS.BAS → RCade: Game Overview

Understanding the original game mechanics and how they map to arcade controls

Original Game Flow

GORILLA.BAS Game Loop
1. SETUP
- Enter player names
- Choose # of rounds
- Set gravity (default 9.8)
2. EACH ROUND
- Generate random cityscape
- Place gorillas on buildings
- Randomize wind speed/direction
3. EACH TURN
- Player enters ANGLE (0-360°)
- Player enters VELOCITY (power)
- Banana flies with physics
- Check: hit building? gorilla? sun? off-screen?
4. RESOLUTION
- Explosion animation
- Victory dance
- Score update → next round
Physics Model:

x = startX + (Vx * t) + (0.5 * wind * t²)

y = startY - (Vy * t) + (0.5 * gravity * t²)

Where:
• Vx = cos(angle) * velocity
• Vy = sin(angle) * velocity
• Wind affects horizontal
• Gravity pulls down

Hit Detection:
• Pixel-based (POINT function)
• Checks color at banana position
• Building color = building hit
• Gorilla color = gorilla hit
• Sun color = sun reaction

RCade Control Mapping

SPINNER (per player)
Continuous rotation → Angle selection
Visual: arrow rotating around gorilla
A BUTTON (per player)
Hold to charge → Power meter
Release to throw banana
DPAD / JOYSTICK
Menu navigation, name entry
Settings screens only
B BUTTON / START
Confirm selections, start game
Menu interactions
